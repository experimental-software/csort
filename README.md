# csort
`csort` is a Unix command-line tool for sorting numbers. It is written in the C programming language for acadamic purposes.

## Usage
```
$ csort 7 3 6 4 1
1 3 4 6 7
$ csort -r 7 3 6 4 1
7 6 4 3 1
$ csort -a bubble_sort 7 3 6 4 1
1 3 4 6 7
```
